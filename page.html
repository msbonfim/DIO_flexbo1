Flex-Wrap

Esta propriedade define a quebra de linha para os itens. Por padrão os itens não vêm com quebra de linhas, assim estes são compactados dentro do conteiner até não haver mais espaço e extravasar, como visto no segundo exemplo, este é o "nowrap".

A correção utilizando a quebra de linha é feita com "flex-wrap: wrap". Porém essa função sozinha gera problemas de espaçamento dentro do conteiner.

Flex-Flow

Funciona como atalho para as propriedades "flex-direction" e "flex-wrap", porém essa função não é muito utilizada.
